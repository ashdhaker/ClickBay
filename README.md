

# ClickBay 

## 
too run app
---> go to folder 
---> open terminal 
---> run command

flutter run -d chrome --web-renderer html --no-sound-null-safety => Web App

flutter run --no-sound-null-safety =>Android App

###### Enable users to connect with each other , sharing photos, and engaging with posts and comments.

## Features

- User interfaces with features like image uploading, user profiles,
timelines, and likes/comments
- Implemented user interaction features like following/followers
- Implemented Search Feature for finding users

### Technologies Used

- Flutter ,Dart ,Firebase
